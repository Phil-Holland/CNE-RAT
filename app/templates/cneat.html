Run RNA-RNA interaction analysis pipeline

This pipeline attempts to predict RNA-RNA interactions between the CNE sequence, and a given set of query RNA sequences (these are expected to be shorter than the CNE sequence). Each query sequence is treated independently in the analysis - only 2-molecule interactions are considered. Two separate tools are used in RNA-RNA interaction analysis: ViennaRNA and IntaRNA. You may choose which of these tools are used during analysis, or run both.
